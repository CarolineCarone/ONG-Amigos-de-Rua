### Sobre o Projeto
O site ONG Amigos de Rua foi desenvolvido com o objetivo de divulgar o trabalho social da ONG, apresentar seus projetos e possibilitar o cadastro de novos voluntários.
O projeto seguiu boas práticas de desenvolvimento web, GitFlow, versionamento semântico e documentação profissional.

### Tecnologias Utilizadas

•	HTML5 (estrutura semântica)

•	CSS3 (design responsivo)

•	JavaScript (interações e dark mode)

•	Git & GitHub (versionamento)

### Funcionalidades

• Página Principal com apresentação da ONG

• Página Projetos mostrando ações da ONG 

• Página de Cadastro de voluntários

• Dark Mode com toggle 

• Layout totalemte responsivo
